Welcome to Circuitikz

The latest stable release of this package is available on CTAN or as a GitHub release. When a release is recent, please allow a couple of days for CTAN mirrors to synchronize.

If you want to use the development version, you can download the git-version, which tracks the master branch:

  • You can download the latest git-version as a single file here
  • You can download the latest git-version for Context as a single file here
  • The latest compiled documentation can be found here
  • Since 0.9.3+, notice that if you want to use the git version style files, you have to download them from here; simply unzip the file in the same place you put your circuitikzgit.sty file.

Released versions

If you encounter problems with the current version, you can use the following files as a compability fallback. Just copy the file to your local tex-project folder and change the package to \usepackage[]{circuitikzgit} (use the appropiate options). The current used circuitikz version can be shown using the command \pgfcircversion.
